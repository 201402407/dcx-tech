# dcx-tech API
```
  DCX팀 기술담당 Server API
```

## 개발환경
* OS : Windows10
* Framework : Spring Framework(전자정부 egov 프레임워크)
* IDE : Eclipse 2021-03 (4.19.0)
* Language : Java
* Design Pattern : MVCS

## API 목록
* L.POINT Certification
* L.POINT CheckLPoint
* L.POINT useLpointUsingCustNo
* L.POINT use
* L.POINT cancel
